## الأداة المتكاملة لتحسين النظام وتثبيت البرامج

أداة صيانة ذكية ومجانية تعتمد على نظام الأوامر (Batch Script) مخصصة لتحديث البرامج، تثبيت المستلزمات الأساسية، وتنظيف نظام ويندوز من البرامج الزائدة لتحسين الأداء بشكل ملحوظ.

### ✨ المميزات الرئيسية:
* **تحديث التطبيقات بضغطة واحدة:** فحص وتحديث جميع البرامج المثبتة على الجهاز باستخدام أداة `Winget`.
* **تثبيت مستلزمات التشغيل (موصى به):** تنزيل وتثبيت حزم الـ Runtimes الضرورية مثل (Visual C++ x64/x86, .NET Runtime, WebView2) لحل مشاكل وأخطاء تشغيل الألعاب والبرامج.
* **قائمة تثبيت برامج مخصصة:** واجهة سريعة لتثبيت البرامج اليومية والأكثر استخداماً، بالإضافة إلى قائمة **Top 10 Apps** (تحتوي على IDM و Revo Uninstaller) مع ميزة البحث الذكي.
* **تنظيف الويندوز (Debloat):** استدعاء أداة *Chris Titus Tech* الشهيرة لتسريع النظام وإيقاف الخدمات غير الضرورية.
* **تحديث التعريفات:** فحص وإجبار النظام على البحث عن تحديثات برامج التشغيل (Drivers).
* **معلومات تفصيلية عن الجهاز:** عرض مواصفات النظام، المعالج، الرام، تفاصيل كرت الشاشة (GPU)، مساحة التخزين (SSD/HDD)، والأجهزة الملحقة المتصلة (الفأرة، لوحة المفاتيح، الصوت).

### 🚀 طريقة التشغيل والاستخدام:
1. قم بتحميل ملف `setup.bat`.
2. انقر بزر الفأرة الأيمن على الملف واختر **"تشغيل كمسؤول" (Run as Administrator)**.
3. اختر الرقم المقابل للميزة التي تريد تشغيلها من القائمة واضغط `Enter`.

## Integrated System Optimization & App Installer Tool

A smart, lightweight command-line utility (Batch Script) designed to upgrade packages, install essential runtimes, remove Windows bloatware, and analyze hardware specs to boost system performance.

### ✨ Key Features:
* **One-Click App Updates:** Scan and upgrade all installed applications automatically using `Winget`.
* **Essential Runtimes Installer (Recommended):** Download and deploy critical runtimes (Visual C++ Redistributables x64/x86, .NET Desktop Runtime, WebView2) to eliminate missing `.dll` errors.
* **Custom Software Menus:** Fast installer for daily essential tools, plus a dedicated **Top 10 Apps** suite (including IDM, Revo Uninstaller) and an interactive search function.
* **Windows Debloater:** Seamlessly launch the *Chris Titus Tech* Windows Utility to optimize OS and disable background resource hogs.
* **Driver Update Detection:** Force Windows to look for the latest system and motherboard drivers.
* **Advanced System Info:** Get detailed hardware diagnostics regarding your OS, CPU, RAM, Graphics Card (GPU), Storage Health/Partitions, and connected peripherals (Mouse, Keyboard, Audio).

### 🚀 How to Run:
1. Download the `setup.bat` file.
2. Right-click on the file and select **"Run as Administrator"**.
3. Type the number of the feature you want to use from the interactive menu and hit `Enter`.
